# CoupGame

桌遊《Coup（政變）》的研究與（未來）數位化專案。

A research and (future) digital-implementation project for the bluffing card game **Coup**.

## 📖 深度研究報告 Deep-Dive Research

完整的遊戲深度研究（規則與機制、策略與心理戰、機率實算、邊角裁決、變體與數位化考量）：

- **[docs/coup-deep-research.md](docs/coup-deep-research.md)** — 《Coup》深度研究報告（繁中為主，中英對照）

報告涵蓋：

- **規則與機制 Rules & Mechanics** — 元件、設定、行動總表、質疑與反制、影響力與淘汰
- **策略與心理戰 Strategy & Psychology** — 金幣節奏、何時吹牛／何時質疑、針對與人數差異
- **機率實算 Probability** — 以組合數學計算「對手宣稱是否為真」的質疑期望值
- **邊角裁決 Edge Cases / FAQ** — 假 Contessa 雙亡、退費規則、誠實被質疑換牌等
- **變體與擴充 Variants** — Reformation、Inquisitor、國庫儲備等
- **數位化實作考量 Implementation Notes** — 狀態機、隱藏資訊、AI/bot 吹牛建模

## 狀態 Status

目前為研究階段；尚未撰寫遊戲程式碼。

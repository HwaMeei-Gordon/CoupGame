# 亡國模式（Doom Mode）設計筆記

> 亡國模式：大體規則與一般模式完全相同，只是**部分角色會有「特殊角色牌」變體**。
> 牌庫每種角色仍是 3 張，但其中會有一張是特殊變體；該變體**平時等同原角色**，
> 只有在「**宣示該角色 → 被質疑 → 攤牌證實為真**」時，額外發動特殊能力。
> 發動後該牌依一般規則洗回牌庫、改抽一張新牌。

內部模式值：`game.mode === 'kingdom'`（顯示名稱為「亡國模式」）。

---

## 角色 1：國王 King（公爵 Duke 的變體）

- **牌庫組成**：原本 3 張公爵 → 變成 **2 張公爵 + 1 張國王**。
- **平時等同公爵**：國王在所有判定上都算「公爵」。
  - 可宣示公爵：課稅（+3）、阻擋外援。
  - 別人質疑你是公爵時，攤出國王＝**誠實**（國王＝公爵）。
- **特殊能力：強制徵收（被質疑、攤牌證實為國王時觸發）**
  當你宣示公爵、被某人質疑、而你攤出的是**國王牌**（證實誠實）時，除了一般結算外，國王額外發動：
  1. 質疑者（質疑錯的人）一般本來就要**翻一張牌**（失 1 影響力）。
  2. **額外**：國王向**質疑者**徵收 **2 金幣**。
  3. 接著國王再向**質疑者的下家**（座位順序的下一位存活玩家）徵收 **1 金幣**。
     - 若「質疑者的下家」就是國王本人，則此步不發動。
  - 金幣不足時，能收多少收多少（min）。
- **觸發時機**：任何「公爵宣示」被質疑且攤出國王皆觸發
  ——含「課稅」與「以公爵阻擋外援」兩種情境。
- **發動後**：國王牌依一般規則洗回牌庫、改抽新牌（國王可能之後被別人抽到）。

## 角色 2：惡魔 Devil（刺客 Assassin 的變體）

- **牌庫組成**：原本 3 張刺客 → 變成 **2 張刺客 + 1 張惡魔**。
- **平時等同刺客**：惡魔在所有判定上都算「刺客」（可付 3 暗殺；被質疑攤惡魔＝誠實）。
- **特殊能力：奪牌（宣示刺客被質疑、攤惡魔證實為真時觸發）**
  當你宣示刺客（暗殺）、被質疑、攤出的是**惡魔牌**（證實誠實）時，質疑者錯誤質疑要付出代價，
  但結算**與一般刺客不同**：
  1. **奪牌（發生在暗殺殺牌之前）**：質疑者必須翻開一張牌（自選；只剩 1 張則強制那張），
     但這張牌**不進質疑者的棄牌堆**，而是**直接成為惡魔持有者的手牌**——
     **眾人皆知**惡魔者手上多了這張公開牌。
  2. **惡魔洗回補抽**：攤出的惡魔牌依例洗回牌庫、徹底洗牌，持有者再**補抽一張**取代（身分重新隱藏）。
  3. **再進行暗殺**：上述奪牌與補抽完成後，才結算原本的暗殺（質疑者／目標再失 1 影響力，此張進棄牌堆）。
- **手牌張數**：持有者原本 2 張 → 攤回惡魔（−1）＋ 補抽（＋1）＋ 奪得對手 1 張（＋1）＝ **3 張**。
- **質疑者只剩 1 張時**：那張仍被奪入惡魔者手牌（不進棄牌堆），質疑者**立即出局**。
- **實作**：`devilSteal(claimant, challenger)`——質疑者選牌 → 推入持有者手牌 + 公開記錄（`k:'devil'`）；
  惡魔牌 `push` 回 `deck`、`shuffleDeck()`、`drawRandom()` 補抽；不呼叫一般 `loseInfluence`（避免該牌進棄牌堆）。

## 角色 4：內奸 Mole（大使 Ambassador 的變體）

- **牌庫組成**：原本 3 張大使 → 變成 **2 張大使 + 1 張內奸**。
- **平時等同大使**：可換牌（exchange）、擋偷；被質疑攤內奸＝誠實。
- **特殊能力：奪牌換內奸（宣示大使被質疑、攤內奸證實為真時觸發）**
  當你宣示大使、被某人質疑、攤出的是**內奸牌**（證實誠實）時：
  1. **質疑者交一張牌給你**（質疑者自己決定給哪張；若只剩 1 張則強制給那張）。你收下。
  2. **你把內奸交給質疑者**。
  3. 質疑者質疑失敗 → 照常**失 1 影響力**：從手上（現含你給的內奸）**自選一張公開棄掉**。
     - 若質疑者原只剩 1 張 → 交牌後手上只有內奸 → 被迫公開內奸 → 出局。
  4. 若這是「換牌（exchange）」行動，接著**照常發動大使換牌效果**（抽 2 張、保留原影響力張數）。
- **與其他變體不同**：內奸**不洗回牌庫**，而是**轉移到質疑者手上**（之後可能被對方棄掉或留用）。
- 淨效果：你用內奸換到質疑者一張牌、對方又失 1 影響力；你（若為換牌行動）再正常換牌。

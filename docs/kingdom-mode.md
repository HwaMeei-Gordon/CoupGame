# 《王國暗戰》Hidden Kingdom — 卡牌・蓋牌喊行動・全員可質疑（v2）

獨立於 Coup 主遊戲的隱藏陣營推理模式。程式 `web/js/kingdom.js`（引擎 + 推理 AI + UI 自含），
首頁「⚔️ 王國暗戰」進入。

## 規則
- **四人各為一座城堡**，各自擁有：**民心(=血，歸 0 被人民起義處死)、金幣、軍隊**。
- **身分秘密(連隊友都不知)**：👑王子×1、🛡忠臣×1、🗡叛徒×2 → 王室 vs 叛徒。
- **起始**：民心 6、金幣 2、軍隊 0、手牌 2 張。資源上限皆 10。
- **牌庫(各 2 張)**：課稅/徵兵/建設/支援/出征/造謠/換牌（共 14）。**收成**為「萬用·不可質疑」(任何牌都可當收成)。

| 牌 | 效果 | 需求 |
|---|---|---|
| 🌾收成(萬用·不可質疑) | 自己 金幣+1、民心+1 | — |
| 💰課稅 | 自己 金幣+3、民心−1 | — |
| ⚔️徵兵 | 自己 軍隊+2、民心−1、金幣−1 | 金幣≥1 |
| 🏛建設 | 自己 金幣−2、民心+2 | 金幣≥2 |
| 🤝支援(對象) | 自己 軍隊−1、民心+1；對方 軍隊+1 | 軍隊≥1 |
| 🔥出征(對象) | 自己 軍隊−3；對方 軍隊−3（**不足→斬殺**） | 軍隊≥3 |
| 🗣造謠(對象) | 自己 金幣−2；對方 民心−2（**到0→起義死亡**） | 金幣≥2 |
| 🔀換牌 | 抽2選2，**並可再出一張**(不可再換牌) | — |

## 流程
- 開局抽 2 張。輪到你 → **蓋一張牌、喊出行動(可與牌不符＝吹牛)**、抽 1 張補手。
- **所有人可質疑**：
  - 質疑**成功**(出牌者在吹牛) → 出牌者 民心−2 金幣−2、**行動取消**、質疑者 民心+2。
  - 質疑**失敗**(出牌者為真) → 行動照常、質疑者 民心−2 金幣−2。
  - 收成不可質疑。首位質疑者結算。

## 王子公開身分（comeback）
- **忠臣死亡後**，王子可選擇於自己回合**公開身分**：民心+2、金幣+2、軍隊+2，此後身分公開（明著打）。
- 一次性、且必須忠臣已死才能發動。

## 勝利
- **叛徒**：讓王子死亡。**王室(王子+忠臣)**：讓兩名叛徒都死亡。

## 平衡（模擬調校）
3000 場 AI 對戰：**王室 51% / 叛徒 49%**，平均 ~53 回合，公開身分發生於 ~48% 局。
勝負核心：叛徒能否在王室(靠忠臣獵殺+王子公開翻盤)剷除兩叛徒前找出並殺掉王子。

## AI（推理）
`KAI` 從公開行動史推估各人攻擊性(誰在造謠/出征→疑似叛徒/敵人)：
- 王子：以「活下去」為要(顧高民心、薄護軍)，忠臣死後傾向公開拿 buff。
- 忠臣：建軍備戰、主動獵殺最具攻擊性者(疑似叛徒)。
- 叛徒：找出王子(優先已公開者)並造謠/出征致死；偶爾吹牛。
- 質疑：行動越威脅自己/隊友、且越多疑時越會質疑(但錯了要 −2/−2，故保守)。

## 待補
- 5/6 人陣營已支援，平衡只調過四人。
- AI 偽裝/釣魚、語言喊話/結盟層、更深的王子身分推理仍可加強。

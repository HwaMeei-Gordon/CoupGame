# 《王國暗戰》Hidden Kingdom — 隱藏陣營推理模式（雛形）

獨立於 Coup 主遊戲的新玩法。程式在 `web/js/kingdom.js`（引擎 + 推理 AI + UI 一檔自含），
首頁「⚔️ 王國暗戰」進入。

## 規則
- **身分(秘密，連隊友都不知道)**：👑王子×1、🛡忠臣×1、🗡叛徒×2（四人）。
  - 隊伍：**王室**(王子+忠臣) vs **叛徒**(兩人)。
- **共用資源 民心**：公開、全場一條，起始 5，範圍 0~10。
- **個人資源**：金幣、士兵(兼護盾與武器)。起始 金幣 2、士兵 1。
- **每回合擇一行動**：
  | 動作 | 效果 |
  |---|---|
  | 🌾 維持民生 | 民心 **+2** |
  | 🪙 收入 | 金幣 +1 |
  | 💰 課稅 | 金幣 +2、民心 **−1** |
  | ⚔️ 招募 | 花 1 金 → 士兵 +1 |
  | 🔥 出兵 | 削目標 1 兵；**目標無兵則破城斬殺**（翻開陣營）；民心 −1 |
- **勝利（整隊一起贏，引擎自動判定）**：
  - 王室：**民心衝到 10** 或 **兩名叛徒全被剷除**
  - 叛徒：**王子被殺** 或 **民心壓到 0**
- 死亡即**公開陣營**（推理燃料）。

## 核心張力
王子拉民心=暴露自己；忠臣知道該拉民心、護王、殺叛徒，但**不知誰是王子**（怕誤殺王子＝叛徒贏）；
叛徒從「誰一直拉民心」找出王子來殺。人人可假裝立場（叛徒偶爾維持民生洗白、王子偶爾收手藏拙）誤導對手。

## 平衡（模擬調校）
4000 場 AI 對戰：**王室 47% / 叛徒 53%**，平均 ~18 回合。主要由「叛徒能否在王室把民心衝到 10 前找出並殺掉王子」決勝。
可調參數（`KGame` opts）：`maintainGain`(預設2)、`taxDrop`(1)、`warDrop`(1)、`startMorale`(5)、`startSoldiers`(1)。
重跑平衡：見 `web/js/kingdom.js` 的 `Coup.Kingdom.{Game,AI}`，可用 headless 迴圈掃參數。

## AI（推理）
`KAI` 從公開行動史推估每位玩家的「王室傾向」(維持民生↑/課稅戰爭↓) 與「王子嫌疑」(最常維持者)，
依自身陣營行動：王子自保+衝民心、忠臣誘餌+獵叛徒、叛徒壓民心+獵王子並偶爾洗白偽裝。

## 待補（雛形之後）
- 5/6 人陣營分配已支援（多 1 忠臣/叛徒），但平衡只調過四人。
- AI 的「立場喊話/結盟」對話層（目前靠行為推理，沒有語言層）。
- 更深的偽裝與反偽裝（王子主動釣叛徒、忠臣假扮王子吸引火力等）。
